My homelab setup running on an Unraid server, featuring:

- **Plex** — Media server with full *arr stack (Radarr, Sonarr, Jackett)
- **AdGuard Home + Unbound** — Network-wide ad-blocking and DNS
- **Home Assistant** — Smart home automation
- **Gitea** — Self-hosted Git server
- **Cloudflare Tunnels** — Secure external access to services
